CSV 格式说明

CSV 文件需包含以下列：

  • sample_id / id - 样本编号（需已存在）
  • label / label_key - 标签键或标签文本
  • label_text - 标签文本（可选，优先匹配 label）
  • comment / note - 标注备注（可选）

sample_id,label,comment
S001,positive,表达了明显的正面情绪
S002,negative,用户有较强抱怨

⚠️ 系统会自动检测：
1. 未知标签：若标签不在方案中，该条会被跳过，不入库
2. 缺失样本：若样本编号不存在，会跳过该条
3. 重复标注：同标注员对同样本的重复标注会更新而非新增
